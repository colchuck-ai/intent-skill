# intent-skill

An [Agent Skill](https://agentskills.io/home) that guides AI assistants to structure product and engineering work using the **Intent** framework: jobs, outcomes, risks, and requirements on the product side; architecture and components on the engineering side; plus decision and change records so intent stays traceable over time, with workflows that separate read (load context, no document changes) from create, update, and delete (including a three-session draft, judge, and review flow for modifications).

The full rules, element definitions, file paths, and templates live in [`intent/SKILL.md`](intent/SKILL.md).
